# cerebro-timezones

> [Cerebro](http://www.cerebroapp.com) plugin to show time in different time zones

![](screenshot.gif)

## Usage

Just type `time` or `время` and name of the city that you want to check.

## Example queries
`time madrid`
`time in sydney`
`время сан франциско`
`время в лиссабоне`
