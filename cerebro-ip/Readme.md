# cerebro-ip

> [Cerebro](http://www.cerebroapp.com) plugin to show your local and external IP addresses

![](screenshot.png)


## Usage

In Cerebro, type `ip`  to see your external and local IPs. Use <kbd>⌘+C</kbd> or <kbd>↩</kbd> to copy IP to clipboard.
